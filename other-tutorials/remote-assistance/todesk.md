# ToDesk

### 简介：

ToDesk 是一款国产个人免费的跨平台远程控制软件。

### 官网：

{% embed url="https://www.todesk.com/" %}

### 使用教程：

以 Windows 平台做示范。

1. 下载对应平台并安装，打开 ToDesk 应用。
2. 将〔您的 ID〕与〔零时密码〕通过复制的方式发送给我们。
